# 🛠️ Windows SOS – Manuale Salvavita per Windows 11

Benvenuto nel progetto **Windows SOS**, il primo manuale interattivo e modulare per risolvere gli errori più comuni di Windows 11. Pensato per utenti di ogni livello, questo strumento ti guida passo dopo passo con comandi, suggerimenti e soluzioni testate.

---

## 📦 Versioni disponibili

- **Versione 1.0 (Base)** – Gratuita e open source  
  Include i moduli principali: BSOD, rete, audio, aggiornamenti, accesso bloccato

- **Versione Estesa (Premium)** – Contattami per riceverla  
  Include moduli aggiuntivi: PC lento, stampante, disco pieno, manutenzione automatica
